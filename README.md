# 単語 Memorize Words with Ebbinghaus Curve

## Backend

```bash
python3 -m venv venv

# Linux
venv/bin/pip install -r requirements.txt
bin/flask run -h 127.0.0.1 -p 5000

# Windows
venv\Scripts\pip install -r requirements.txt
bin\flask run
```

## Frontend

```bash
yarn install
yarn start
```

Open with [Expo](https://expo.io/tools).
